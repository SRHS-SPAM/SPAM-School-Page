### essential Skills

1. Media Queries

   ```css
   @media (max-width: 600px) {
   }
   ```

2. Flexible Units

   - (%), em, rem, vh, vw

     | 절대적인 값 | 상대적인 값 |
     | :---------: | :---------: |
     |     cm      |     em      |
     |     mm      |     rem     |
     |     px      |     vw      |
     |             |     vh      |
     |             |      %      |

3. Grid System
4. Flexible Layout
5. Viewport

   ```html
   <meta name="viewport" content="width=device-width, initial-scale=1.0" />
   ```

6. HTML Semantic Tag
